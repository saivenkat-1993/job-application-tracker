# Job Application Tracker

A full-stack web application built with ASP.NET Core 8, Entity Framework Core, and Blazor to help developers track their job applications, interviews, and contacts in one place.

## Features

- Track job applications with status (Applied, Interview, Offer, Rejected)
- Manage contacts and recruiters per application
- Schedule and log interviews
- Dashboard with application stats and response rates

## Tech Stack

- **Backend:** ASP.NET Core 8 Web API
- **Frontend:** Blazor WebAssembly
- **Database:** PostgreSQL with Entity Framework Core
- **Auth:** JWT Authentication
- **Testing:** xUnit, Moq
- **CI/CD:** GitHub Actions
- **Deployment:** Azure App Service
